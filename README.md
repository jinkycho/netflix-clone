# 넷플릭스 클론

- React를 기반으로 UI/UX가 잘 구성된 넷플릭스를 분석하여 클론 웹사이트를 개발했습니다.
- TMDB API를 이용해 컨텐츠가 다양한 웹사이트를 구성했습니다.
- 웹/모바일에서 반응형으로 화면을 구현하여 넷플릭스 사이트와 유사하게 구성했습니다.
